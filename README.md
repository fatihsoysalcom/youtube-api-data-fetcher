# YouTube API Data Fetcher

This Python script demonstrates how to use the YouTube Data API v3 to search for videos. It fetches and displays titles, channels, and URLs for relevant STEM educational content, illustrating how platforms can programmatically access and curate YouTube resources.

## Language

`python`

## How to Run

1. Obtain a YouTube Data API v3 key from Google Cloud Console.
2. Set the API key as an environment variable: export YOUTUBE_API_KEY='YOUR_API_KEY'
3. Run the script: python youtube_data_fetcher.py
